Tree with cascading checkboxes. Indeterminate state is also supported.

Notes:

  • checkbox is bound to a native record's field, so they restore their state when you switch to other examples and back
  • checkbox's bindings are unidirectional
  • the state of checkbox hierarchy is not validated in the data layer (such validation would be extremely hard to implement and maintain). Instead, the touched part of hierarchy is recalculated in controller.
